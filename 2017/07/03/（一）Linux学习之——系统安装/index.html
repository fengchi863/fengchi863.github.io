（一）Linux学习之——系统安装

发表于 2017-07-03 | 分类于 linux
  1. 在Linux系统中，每个设备都被当成一个文件来对待。
  2. IDE硬盘对应的文件名为：/dev/hd[a-d]
    SCSI/SATA/USB硬盘对应的文件名为：/dev/sd[a-p]
  3. IDE1(Primary)包含Master和Slave两个，分别为/dev/hda和/dev/hdb
    IDE2(Secondary)包含Master和Slave两个，分别为/dev/hdc和/dev/hdd
  4. 根据Linux内核检测到磁盘的顺序决定设备的文件名
  5. 磁盘的第一个扇区主要记录了两个重要信息，分别是：主引导分区MBR和分区表
  6. 分区表只有64bytes而已，最多只能容纳四个分区，这四个分区被称为主和扩展分区
  7. 逻辑分区的设备名称号码由5号开始，P1:/dev/hda1 L1:/dev/hda5
  8. 整个开机流程到操作系统之前的动作应该是这样的：BIOS（开机主动执行的韧体，会认识第一个可开机的设备）、MBR（内含引导加载程序）、引导加载程序（Boot loader）、内核文件（开始操作系统的功能）
  9. Boot loader的功能：提供菜单（使用户可以选择不同的开机选项）、载入内核文件（直接指向可开机的程序区段来开始操作系统）、转交其他loader（将引导加载功能转交给其他loader负责）
  10. 开机需要引导加载程序，引导加载程序可以安装在MBR以及启动扇区这两处
  11. 挂载mount就是利用一个目录当成进入点
  12. 最简单的分区方法就是仅分出根目录/与内存交换空间/&swap即可
